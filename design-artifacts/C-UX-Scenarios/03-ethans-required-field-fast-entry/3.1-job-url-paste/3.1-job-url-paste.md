# 3.1 Job URL Paste

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | Ethan's Required-Field Fast Entry |
| **Page Number** | 3.1 |
| **Platform** | Desktop |

## Overview

**Page Purpose:** Let Ethan start a fast application entry by pasting a job URL and immediately seeing what the system can prefill.

**Entry Context:** Ethan is batch-adding roles and begins from the add form with a copied job URL.

**Exit Action:** Ethan submits the URL and moves to `Required Fields Prompt` after extraction/validation runs.

**On-Page Interactions:**
- Paste job URL into input field
- Trigger parse/extract action
- View parsing progress and initial prefills

## First-Step Context (Q3 + Q4 + Q6)

**User Situation:** Ethan is adding several jobs quickly and wants to avoid spending extra time on repeated manual entry.

**Mental State:**
- **Hope:** Each job position includes all required data.
- **Worry:** The record will be incomplete and later forgotten.

**Discovery Method:** Ethan arrives by starting a new application and pasting a copied job URL into the add form.
