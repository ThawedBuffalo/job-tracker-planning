## Scenarios

### [01: Priya's Fast Job Entry](01-priyas-core-pipeline-momentum/01-priyas-core-pipeline-momentum.md)
**Persona:** Priya — Keep momentum with low-friction updates
**Pages:** Home, Dashboard, Add Application, Confirm Saved
**User Value:** Priya can get into the system fast and enter a usable job record with minimal typing.
**Business Value:** Faster capture supports operational simplicity and sustained active usage.

---

### [02: Clara's Follow-Up Confidence Loop](02-claras-follow-up-confidence-loop/02-claras-follow-up-confidence-loop.md)
**Persona:** Clara — Step-by-step clarity for unfamiliar stages
**Pages:** Email Link Entry, Application Detail, Next-Step Guidance, Confirm Next Action
**User Value:** Clara sees the current stage clearly and knows the next required action without ambiguity.
**Business Value:** Better follow-through reduces missed stages and encourages repeat system usage.

---

### [03: Ethan's Required-Field Fast Entry](03-ethans-required-field-fast-entry/03-ethans-required-field-fast-entry.md)
**Persona:** Ethan — Reduce overhead per entry and update
**Pages:** Job URL Paste, Required Fields Prompt, Complete Missing Fields, Save Application
**User Value:** Ethan can enter data quickly without missing essential fields.
**Business Value:** More complete records improve consistency and increase repeat workflow usage.

---

### [04: Ravi's Unified Global Entry](04-ravis-unified-global-entry/04-ravis-unified-global-entry.md)
**Persona:** Ravi — Manage global opportunities in one consistent system
**Pages:** Job URL Paste, Parsed Data Review, Complete Required Fields, Save Application
**User Value:** Ravi can bring opportunities from external job sites into one workflow with minimal friction.
**Business Value:** Unified intake helps increase job submittals by reducing fragmentation across sources.
