# 4.1 Job URL Paste

## Page Metadata

| Property | Value |
|----------|-------|
| **Scenario** | Ravi's Unified Global Entry |
| **Page Number** | 4.1 |
| **Platform** | Desktop |

## Overview

**Page Purpose:** Let Ravi start a unified application entry from any external job site by pasting a job URL into the app.

**Entry Context:** Ravi is browsing multiple job sources and wants to move one promising opportunity into his central workflow without repetitive manual entry.

**Exit Action:** Ravi submits the URL and moves to `Parsed Data Review` after extraction begins.

**On-Page Interactions:**
- Paste job URL into intake field
- Trigger parse/mapping action
- View extraction progress and initial prefill summary

## First-Step Context (Q3 + Q4 + Q6)

**User Situation:** Ravi is switching between multiple job sources and wants one workflow that can absorb opportunities smoothly.

**Mental State:**
- **Hope:** Move easily between websites for positions and enter data without friction.
- **Worry:** The app feels clunky and slows him down.

**Discovery Method:** Ravi arrives by copying a job URL from another site and pasting it directly into the app's add flow.
